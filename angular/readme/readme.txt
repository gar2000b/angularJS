angularJS
---------

SPA (Single Page Application) Framework
Multiple Views on a single page.
Challenges with SPA - Routing, Ajax/Promises
One core library - maintenance :)
Two way data-binding, MVC, Routing, Testing, jqLite (mini-me JQuery for DOM manipulation), templates, view model, controllers, validation, dependency injection

angularjs.org
download the js file
link it to website:
<script src="angular.min.js"></script>

Directives, Filters and Data Binding.

Directives:
Teach HTML new tricks.
<html ng-app> - angular directive.
...
Name: <input type="text" ng-model="name" /> {{ name }} - data binding expression.

Modules: config -> routes -> View && Controller
Modules are Object Containers.
